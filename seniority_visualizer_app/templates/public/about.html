About

This is pet project of mine I started in late 2019. I hurt my back and was off the line for a few months, so I picked up a few programming languages and threw together a few scripts for my own use that kept track of some fancy seniority stuff. But, I wanted to learn web development so I could deploy them as something useful, and this is the result of that venture. I will continuously be adding features and new data views to the project as I continue with it.

If you know Python, Pandas, Flask, JavaScript, I'd be happy to take a look at any pull requests you may have. Even if you don't, please feel free to message me about any bugs or features you'd like to see implemented.
